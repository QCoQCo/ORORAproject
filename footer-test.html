푸터 컴포넌트 테스트

arataBUSAN 프로젝트의 푸터 컴포넌트가 성공적으로 로드되고 있는지 확인하는 테스트 페이지입니다.
아래로 스크롤하여 푸터의 모든 기능을 확인해보세요.

🎨 모던 디자인

그라데이션 배경과 현대적인 카드 스타일로 디자인된 푸터

📱 반응형 레이아웃

다양한 디바이스에서 최적화된 반응형 디자인

🔗 빠른 링크

사이트 주요 페이지로의 편리한 네비게이션 링크

📱 소셜 링크

부산 관련 소셜 미디어 링크 (인스타그램, 유튜브)

✨ 인터랙티브 효과

호버 효과와 스크롤 애니메이션이 적용된 인터랙티브 요소

🔄 자동 업데이트

현재 연도 자동 업데이트 및 스크롤 투 탑 기능

💡 테스트 방법

1. 아래로 스크롤하여 푸터를 확인해보세요
2. 푸터의 각 링크들이 올바르게 작동하는지 확인해보세요
3. 푸터 로고를 클릭하면 페이지 상단으로 스크롤됩니다
4. 소셜 링크 클릭 시 콘솔에 로그가 출력됩니다
